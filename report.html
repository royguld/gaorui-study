    📧 自动邮件设置 一次性配置，之后孩子点“发送报告”即自动发到邮箱

    报告会同时发送到 gaojh@chinanewbase.com 和 royguld@gmail.com（收件人在下面的脚本里设定，可改）。

    ▶ 第一次设置：3 分钟部署你的免费邮件脚本（点开看步骤）
    1. 用 royguld@gmail.com 登录，打开 script.google.com，点“新建项目”。
    2. 把下面全部代码粘贴进去，覆盖原有内容，保存。
    3. 点右上角“部署” → “新建部署” → 类型选“Web 应用”；“执行身份”选我(你自己)；“谁可以访问”选任何人 → 部署。
    4. 首次会要求授权，允许即可（脚本只用来替你发邮件）。
    5. 复制生成的 Web 应用网址（以 /exec 结尾），粘贴到上面输入框，点“保存”，再点“发送一封测试报告”验证。
    function doPost(e) {
  var data = JSON.parse(e.postData.contents);
  // 收件人：两个邮箱都发；想改就改这一行
  var TO = "gaojh@chinanewbase.com, royguld@gmail.com";
  var subject = data.subject || (data.childName + " 学习报告 " + data.date);
  var html = data.html || "";
  var pdf = Utilities.newBlob(
    '<meta charset="utf-8">' + html, "text/html", "report.html"
  ).getAs("application/pdf").setName((data.childName || "孩子") + "-学习报告-" + data.date + ".pdf");
  MailApp.sendEmail({
    to: TO,
    subject: subject,
    htmlBody: html,
    attachments: [pdf]
  });
  return ContentService.createTextOutput("ok");
}

    脚本用你的 Gmail 发送，把网页报告转成 PDF 作为附件；不改代码就同时发给两个邮箱。收件方每天会收到一封带 PDF 附件的邮件。
